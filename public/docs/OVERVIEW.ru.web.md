# BugIt QA Agent — обзор

BugIt — это коммерческий агент VS Code Copilot, который превращает черновые заметки о тестировании в единообразные отчёты об ошибках. Он готовит черновики локально в вашем рабочем пространстве и записывает данные в подключённые сервисы только после предпросмотра и подтверждения.

## Основной рабочий процесс

- Фиксируйте черновые шаги воспроизведения, логи, скриншоты и ожидаемое поведение.
- Составляйте структурированный отчёт с заголовком, серьёзностью, окружением, шагами и доказательствами.
- Ищите возможные дубликаты в подключённом трекере.
- Проверяйте и подтверждайте место назначения и итоговое содержимое перед любой внешней записью.
- Добавляйте комментарии о верификации после повторной проверки исправления.

## Конфиденциальность и контроль

- BugIt не отправляет в Taskivator никакую продуктовую аналитику или телеметрию тикетов.
- Ваш подключённый ИИ-провайдер и включённые интеграции обрабатывают только то содержимое, которое вы решаете им отправить.
- Запросы на лицензию и обновления используют данные лицензии и односторонний идентификатор устройства, а не содержимое тикетов.
- Режим пробного запуска (dry-run) не позволяет встроенным помощникам Python выполнять запись, но вы всё равно должны проверять внешние действия MCP.
- Файлы конфигурации никогда не должны содержать значений учётных данных.

## Уровни интеграции

- С сопровождением: Jira Cloud и Confluence Cloud через Atlassian Rovo MCP.
- Публичный предпросмотр с сопровождением: Azure DevOps через удалённый MCP-сервис Microsoft.
- Экспериментальные с проверкой в реальном времени: Sentry, GitHub, Linear и Notion.
- Только рекомендации по настройке: предоставленные организацией совместимые серверы для других трекеров, инструментов для сбоев, управления тестированием, коммуникаций и сервисов знаний.
- Не поддерживаются автоматической настройкой: коннекторы к хранилищам S3, Google Drive и Azure Blob.

## Область выпуска

- BugIt — это текущий опубликованный коммерческий выпуск, который активно поддерживается.
- Windows 11, VS Code, GitHub Copilot и Python 3.10–3.13 — это среда, для которой сертифицирован выпуск.
- Полное руководство пользователя и обзор доступны в виде PDF на английском и на каждом поддерживаемом языке — просмотрите или загрузите их ниже.
