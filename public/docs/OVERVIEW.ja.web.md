# BugIt QAエージェント — 概要

BugItは、雑多なテストメモを一貫性のあるバグレポートに変換する商用のVS Code Copilotエージェントです。まずワークスペース内でローカルに下書きを作成し、プレビューと確認を経た後にのみ、連携先のサービスへ書き込みます。

## 基本ワークフロー

- 大まかな再現メモ、ログ、スクリーンショット、期待される挙動を取り込みます。
- タイトル、重大度、環境、手順、証跡を含む構造化されたレポートを下書きします。
- 連携中のトラッカーを検索し、重複の可能性がある項目を洗い出します。
- 外部への書き込みを行う前に、書き込み先と最終的な内容をプレビューして承認します。
- 修正の再テスト後に、検証コメントを追加します。

## プライバシーと制御

- BugItは、製品分析データやチケットのテレメトリをTaskivatorに送信しません。
- 連携中のAIプロバイダーおよび有効化した各連携は、あなたが送信を選択したコンテンツのみを処理します。
- ライセンスおよび更新に関するリクエストでは、チケットの内容ではなく、ライセンスデータと一方向のデバイス識別子を使用します。
- ドライランモードは、同梱のPythonヘルパーによる書き込みを防ぎますが、外部のMCPアクションは引き続き必ず確認する必要があります。
- 設定ファイルには、資格情報の値を決して含めてはなりません。

## 連携ティア

- ガイド付き: Atlassian Rovo MCPを介したJira CloudおよびConfluence Cloud。
- ガイド付きパブリックプレビュー: MicrosoftのリモートMCPサービスを介したAzure DevOps。
- ライブ検証付きの実験的機能: Sentry、GitHub、Linear、Notion。
- セットアップ手引きのみ: その他のトラッカー、クラッシュ解析ツール、テスト管理、コミュニケーション、ナレッジサービス向けに組織が提供する互換サーバー。
- 自動セットアップ非対応: S3、Google Drive、Azure Blobのストレージコネクタ。

## リリース範囲

- BugItは現在公開されている商用リリースであり、継続的に保守されています。
- Windows 11、VS Code、GitHub Copilot、およびPython 3.10から3.13までが、リリース認定済みの動作環境です。
- 完全なユーザーガイドと概要は、英語およびすべての対応言語のPDFとして提供されています。下記からプレビューまたはダウンロードできます。
