# 退款政策

BugIt 是 Taskivator 提供的授权产品。本政策说明在哪些情况下可以退款。

## 7 天退款期

如果符合以下任一情况，你可以在购买后 7 天内申请退款：

- BugIt 未按宣传的方式工作
- 可复现的技术问题导致无法正常使用
- 支持团队无法在合理时间内解决问题

我们可能会请你简要说明问题以及你已尝试的步骤，以便支持团队确认问题或先协助你解决。

## 哪些情况不予退款

以下情况通常不予退款：

- 改变主意
- 误购
- 未满足所记载的系统要求
- 超过 7 天期限后提交的申请

购买前请查看要求：Visual Studio Code、一个 AI 提供方（GitHub Copilot 或你自己的 OpenAI 或 Anthropic 密钥）以及受支持的 Python 版本。BugIt 不含 AI 模型或订阅。

## 退款获批后会发生什么

当我们批准退款时，购买将被撤销，相关授权将被停用：

- 相关授权将被永久吊销
- 使用该授权的后续激活将停止
- 该授权的后续更新将停止
- 与该授权关联的支持将结束

## 如何申请退款

请在 7 天内使用购买时所用的账户发送邮件至 [support@bugit.dev](mailto:support@bugit.dev)，并描述你遇到的问题。请勿包含机密源代码、客户数据、私有工单或密钥。我们会通过邮件确认处理结果。

本政策仅涉及退款，不限制你依据适用消费者保护法所享有的任何权利。
