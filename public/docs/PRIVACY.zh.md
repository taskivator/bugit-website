以通俗语言概述本软件收集哪些信息、不收集哪些信息。一切都在您自己的设备上运行。

## 本软件向 Taskivator 发送的内容

本软件向我们发送的唯一内容是**许可证/更新数据**：

- 您的**许可证密钥**，
- 一个**匿名的、经单向哈希处理的设备指纹**——一个由基本机器属性派生而来的
  16 位字符哈希值。它无法被逆推以识别您本人或您的硬件，以及
- **仅当您在首次运行设置时进行了设置**，您所选择的一个简短席位标签，以便区分
  Team 许可证下的各个席位（例如姓名、用户名或邮箱——从不要求为真实信息，
  也从不进行验证）。如果您未进行设置，则此项根本不会被发送。

这些内容仅发送至 Taskivator 许可证服务器，且仅用于激活/验证您的席位以及检查
是否有更新版本可用。**没有遥测，没有分析，没有跟踪，没有广告——永远如此。**

## 完全保留在您设备上的内容

- 您的规格说明、术语表、团队风格以及习得的更正
- 您的 `config.json` 及本地项目文件
- 您的 API 令牌（保存在您操作系统的凭据库中）

以上内容均不会被传输到任何地方。

## 仅发送至*您*所连接服务的内容

为撰写和提交工单，您的报告文本会被发送至您所使用的 AI 模型（GitHub Copilot，
或您自己的 OpenAI/Anthropic 密钥）以及您提交到的 tracker（如 Jira 或 Azure
DevOps）。这些是**您**自行选择并连接的 AI 与工具——它绝不会经由 Taskivator
转发、复制到 Taskivator，或被 Taskivator 查看。

## 凭据

API 令牌存放在您操作系统的凭据库中——绝不会存入文件，也绝不会传输给
Taskivator。

## 网站分析

bugit.dev 网站使用注重隐私的 Cloudflare Web Analytics 服务来统计页面性能和访问量。它不使用 Cookie，也不会跨网站跟踪您。

在您同意后，本网站还会使用 Google 衡量（Google Ads）来评估我们广告的效果。在您于 Cookie 横幅中选择全部接受或开启广告之前，它一直处于关闭状态，您可以随时通过页脚的 Cookie 偏好设置链接更改或撤回您的选择。当您允许时，我们可能会共享某次购买的金额、币种和不含个人信息的订单编号，以便将销售归因于广告。我们绝不会共享您的错误报告内容、您的支付卡信息或您在产品中输入的任何内容。

这仅适用于网站本身。如上所述，本软件不会发送任何分析数据、广告或跟踪。
