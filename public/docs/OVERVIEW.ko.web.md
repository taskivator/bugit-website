# BugIt QA Agent — 개요

BugIt는 정리되지 않은 테스트 메모를 일관된 버그 리포트로 변환하는 상용 VS Code Copilot 에이전트입니다. 작업 공간 내에서 로컬로 초안을 작성하며, 미리 보기와 확인을 거친 뒤에만 연결된 서비스에 기록합니다.

## 핵심 워크플로

- 대략적인 재현 메모, 로그, 스크린샷, 기대 동작을 수집합니다.
- 제목, 심각도, 환경, 재현 단계, 증적을 갖춘 구조화된 리포트 초안을 작성합니다.
- 연결된 트래커에서 중복 가능성이 있는 항목을 검색합니다.
- 외부에 기록하기 전에 대상과 최종 내용을 미리 보고 승인합니다.
- 수정 사항을 재검증한 뒤 확인 코멘트를 추가합니다.

## 개인정보 보호와 제어

- BugIt는 어떠한 제품 분석 데이터나 티켓 텔레메트리도 Taskivator에 전송하지 않습니다.
- 연결된 AI 제공자와 활성화된 통합은 사용자가 전송하기로 선택한 콘텐츠만 처리합니다.
- 라이선스 및 업데이트 요청에는 티켓 내용이 아니라 라이선스 데이터와 단방향 기기 식별자가 사용됩니다.
- 드라이런 모드는 번들로 제공되는 Python 헬퍼의 기록을 방지하지만, 외부 MCP 작업은 여전히 사용자가 직접 검토해야 합니다.
- 구성 파일에는 자격 증명 값이 절대 포함되어서는 안 됩니다.

## 통합 등급

- 가이드형: Atlassian Rovo MCP를 통한 Jira Cloud 및 Confluence Cloud.
- 가이드형 공개 미리 보기: Microsoft의 원격 MCP 서비스를 통한 Azure DevOps.
- 실시간 검증을 동반한 실험적 지원: Sentry, GitHub, Linear, Notion.
- 설정 안내만 제공: 다른 트래커, 크래시 도구, 테스트 관리, 커뮤니케이션, 지식 관리 서비스를 위해 조직에서 제공하는 호환 서버.
- 자동 설정 미지원: S3, Google Drive, Azure Blob 스토리지 커넥터.

## 릴리스 범위

- BugIt는 현재 게시된 상용 릴리스로, 활발히 유지 관리되고 있습니다.
- Windows 11, VS Code, GitHub Copilot, Python 3.10부터 3.13까지가 릴리스 검증 환경입니다.
- 전체 사용자 가이드와 개요는 영어 및 지원되는 모든 언어의 PDF로 제공됩니다 — 아래에서 미리 보거나 다운로드하세요.
