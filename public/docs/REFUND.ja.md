# 返金ポリシー

BugIt は Taskivator が提供するライセンス製品です。本ポリシーでは、購入をどのような場合に返金できるかを説明します。

## 7日間の返金期間

次のいずれかに当てはまる場合、購入から7日以内に返金をリクエストできます:

- BugIt が説明どおりに動作しない
- 再現可能な技術的問題により通常の利用ができない
- サポートが妥当な期間内に問題を解決できない

サポートが問題を確認したり、先に解決をお手伝いしたりできるよう、問題の概要と既にお試しいただいた手順を簡単にお伺いする場合があります。

## 返金の対象外となる場合

次の場合は原則として返金の対象外です:

- 気が変わった場合
- 誤って購入した場合
- 記載されているシステム要件を満たしていない場合
- 7日間の期間を過ぎてからのリクエスト

購入前に要件をご確認ください: Visual Studio Code、AIプロバイダー（GitHub Copilot またはご自身の OpenAI もしくは Anthropic のキー）、対応する Python のバージョン。BugIt には AI モデルやサブスクリプションは含まれません。

## 返金が承認された場合の流れ

返金を承認すると、購入が取り消され、対象のライセンスは無効化されます:

- 対象のライセンスは恒久的に取り消されます
- そのライセンスでの以後のアクティベーションは停止します
- そのライセンスの以後のアップデートは停止します
- そのライセンスに紐づくサポートは終了します

## 返金のリクエスト方法

購入に使用したアカウントから7日以内に [support@bugit.dev](mailto:support@bugit.dev) 宛にメールし、問題の内容をお知らせください。機密のソースコード、顧客データ、非公開チケット、秘密情報は含めないでください。結果はメールでお知らせします。

本ポリシーは返金のみを対象とし、適用される消費者保護法に基づくお客様の権利を制限するものではありません。
