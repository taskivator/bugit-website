## 本ソフトウェアが Taskivator に送信するもの

本ソフトウェアが当社に送信する唯一の情報は、**ライセンス／更新データ**です:

- お客様の**ライセンスキー**、
- **匿名の、一方向でハッシュ化されたデバイスフィンガープリント** — 基本的なマシン属性から導出される 16 文字のハッシュです。これを逆算して、お客様またはお客様のハードウェアを特定することはできません。そして
- **初回セットアップ時に設定した場合に限り**、Team ライセンスのシートを区別できるようにお客様が選んだ、短い席ラベル（例：名前、ユーザー名、またはメールアドレス — 実在するものである必要は一切なく、検証も行われません）。設定しない場合、これが送信されることはありません。

これらは Taskivator のライセンスサーバーにのみ送信され、お客様のシートをアクティベート／検証し、より新しいバージョンが利用可能かどうかを確認する目的にのみ使用されます。**テレメトリ、アナリティクス、トラッキング、広告は、一切行いません。**

## お客様のデバイスに完全にとどまるもの

- お客様の仕様、用語集、ハウススタイル、および学習された修正
- お客様の `config.json` およびローカルのプロジェクトファイル
- お客様の API トークン（OS の資格情報ストアに保管されます）

これらのいずれも、どこにも送信されません。

## *お客様が*接続するサービスにのみ送信されるもの

チケットを作成して起票するために、お客様の報告テキストは、お客様が使用する AI モデル（GitHub Copilot、またはお客様ご自身の OpenAI／Anthropic キー）、および起票先のトラッカー（Jira や Azure DevOps など）に送信されます。これは**お客様**が選択し、接続した AI およびツールであり、Taskivator を経由したり、Taskivator にコピーされたり、Taskivator に閲覧されたりすることは決してありません。

## 資格情報

API トークンは、お客様のオペレーティングシステムの資格情報ストアに保管されます — ファイルに保存されることはなく、Taskivator に送信されることも決してありません。

## ウェブサイトのアクセス解析

bugit.dev ウェブサイトは、プライバシーを重視したサービスである Cloudflare Web Analytics を使用して、ページのパフォーマンスと訪問数を測定しています。Cookie は使用せず、サイトをまたいだ追跡も行いません。これはウェブサイトのみに適用されます。上記のとおり、本ソフトウェア自体はいかなる分析データも送信しません。
