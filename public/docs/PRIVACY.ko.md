## 본 소프트웨어가 Taskivator로 전송하는 것

본 소프트웨어가 당사로 전송하는 유일한 것은 **라이선스/업데이트 데이터**입니다:

- 귀하의 **라이선스 키**,
- **익명의 단방향 해시된 기기 지문** — 기본적인 기기 속성으로부터 도출된 16자리 해시입니다.
  이는 역으로 되돌려 귀하나 귀하의 하드웨어를 식별할 수 없습니다. 그리고
- **최초 실행 설정 시 지정한 경우에만**, Team 라이선스의 좌석을 서로 구분할 수 있도록 귀하가
  선택한 짧은 좌석 이름표(예: 이름, 사용자명 또는 이메일 — 실제일 필요가 전혀 없으며 검증되지도
  않습니다). 지정하지 않으면 이는 전혀 전송되지 않습니다.

이들은 오직 Taskivator 라이선스 서버로만, 그리고 오직 귀하의 좌석을 활성화/검증하고 최신 버전이
있는지 확인하기 위해서만 전송됩니다. **텔레메트리도, 분석도, 추적도, 광고도 결코 없습니다.**

## 전적으로 귀하의 기기에만 남는 것

- 귀하의 사양, 용어집, 하우스 스타일 및 학습된 수정 사항
- 귀하의 `config.json` 및 로컬 프로젝트 파일
- 귀하의 API 토큰(귀하의 OS 자격 증명 저장소에 보관됨)

이 중 어느 것도 어디에도 전송되지 않습니다.

## 오직 *귀하가* 연결한 서비스로만 전달되는 것

티켓을 작성하고 등록하기 위해, 귀하의 보고서 텍스트는 귀하가 사용하는 AI 모델(GitHub Copilot,
또는 귀하 자신의 OpenAI/Anthropic 키)과 귀하가 등록하는 추적기(예: Jira 또는 Azure DevOps)로
전송됩니다. 이는 **귀하가** 선택하고 연결한 AI 및 도구이며 — 결코 Taskivator를 거쳐 라우팅되거나,
Taskivator로 복사되거나, Taskivator에 의해 열람되지 않습니다.

## 자격 증명

API 토큰은 귀하 운영 체제의 자격 증명 저장소에 보관되며 — 결코 파일에 저장되지 않고, 결코
Taskivator로 전송되지 않습니다.

## 웹사이트 분석

bugit.dev 웹사이트는 개인정보 보호를 우선하는 서비스인 Cloudflare Web Analytics를 사용하여 페이지 성능과 방문 수를 측정합니다. 쿠키를 사용하지 않으며 사이트 간 추적도 하지 않습니다. 이는 웹사이트에만 적용됩니다. 위에서 설명한 대로 소프트웨어 자체는 어떠한 분석 데이터도 전송하지 않습니다.
